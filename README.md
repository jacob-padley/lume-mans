# Lume Mans

Live lighting events for the 24 Hours of Le Mans!

Lume Mans is a Tauri app which uses OCR to detect the current status of a WEC race. This state is displayed in the front-end window, and then a lighting event is send out to the Avolites Titan API to trigger a lighting event. The current supported track states are:

- Race Start
- Green Flag
- Yellow Flag
- Slow Zone
- Full Course Yellow / Full Course Yellow Ending
- Safety Car / Safety Car Ending
- Virtual Safety Car / Virtual Safety Car Ending
- Red Flag
- Checkered Flag
